# 概要
Webサイト上にある複数のpdfファイルをまとめてダウンロードするためのスクリプト

# 注意
スクレイピングを使用しているので,Webサイトに負荷をかけない,もしくはマイニングが禁止されているサイトにはスクレイピングを行わないようにしてください.
